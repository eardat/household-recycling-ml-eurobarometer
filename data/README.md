# Data Directory

- `raw/ZA5474_FEB316/` contains the raw Flash Eurobarometer 316 / GESIS ZA5474 workbook.
- `raw/ZA4742_EB68_2/` contains the raw Eurobarometer 68.2 / GESIS ZA4742 SAV file.
- `processed/` preserves processed CSV snapshots used during earlier development.

Raw data files are preserved and were moved, not deleted.
Raw GESIS survey files are not included in this repository and should be downloaded from the GESIS Data Archive.
